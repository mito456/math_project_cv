# Real-Time Webcam Filters using OpenCV

## Overview
This project implements real-time image processing using OpenCV, applying filters such as grayscale, Gaussian blur, and edge detection on webcam frames.

## Features
- Grayscale conversion
- Blur filtering
- Edge detection
- Frame-by-frame processing

## Tech Stack
- Python
- OpenCV
- NumPy

## Note
Webcam is simulated in Google Colab using browser capture due to environment limitations.
